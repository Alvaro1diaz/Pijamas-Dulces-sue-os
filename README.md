# Dulces Sueños Pijamas - Página web

## Archivos incluidos

- `index.html`: estructura de la página.
- `style.css`: diseño visual, colores y versión responsive.
- `script.js`: menú móvil, filtros del catálogo y formulario conectado a WhatsApp.

## Cómo verla en tu computador

1. Descomprime el archivo ZIP.
2. Abre la carpeta `dulces_suenos_pijamas_web`.
3. Haz doble clic en `index.html`.

## Cómo publicar gratis en Netlify

1. Entra a Netlify.
2. Crea una cuenta o inicia sesión.
3. Arrastra la carpeta completa `dulces_suenos_pijamas_web` al panel de publicación.
4. Netlify generará un enlace público.
5. Puedes cambiar el nombre del sitio desde la configuración.

## Cómo publicar gratis en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube `index.html`, `style.css` y `script.js`.
3. Activa GitHub Pages desde Settings > Pages.
4. Selecciona la rama principal.
5. GitHub generará el enlace público.

## Qué debes cambiar después

- Reemplazar las imágenes ilustrativas por fotos reales de las pijamas.
- Ajustar precios exactos por referencia.
- Confirmar tallas disponibles.
- Agregar políticas de cambio, envíos y garantía si las manejas.
- Agregar dominio propio si quieres una página más profesional.
